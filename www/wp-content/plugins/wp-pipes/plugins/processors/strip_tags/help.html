Strip Tags processor for WPPipes is for strip specific HTML tags out of a HTML input.
In this sample, we strip every HTML tags (*) out of [ op[2] fulltext ] field, except: div,p,a,img.
Allowable params for strip tags: a, div, span, embed, object, ... OR * wildcard for all HTML tags. Seperate them by comma.
Input: an HTML field.
Output: a stripped HTML field.
